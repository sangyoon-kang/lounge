상품관리

홈 > 환경설정 > 상품관리

체크사항 (상품관리 및 쇼핑몰 기능을 제공합니다.)

  • - 스킨위치 : /admin/product/skin
  • - 특정 상품분류 열기
       ㆍ방법1 : 생성코드 삽입시 코드($catcode)를 포함합니다. 예)<? $catcode="100000"; include "$_SERVER[DOCUMENT_ROOT]..."
       ㆍ방법2 : 페이지 링크시 분류코드(catcode=100000) 를 추가합니다. 예) /product.php?catcode=1000000
       ㆍ분류코드는 "상품관리 > 상품분류 > 해당 분류 클릭 > 분류코드" 에 나옵니다.
  • - 특정 상품그룹 열기
       ㆍ생성코드 삽입시 상품그룹($grp)를 포함합니다. 예)<? $grp="best"; include "$_SERVER[DOCUMENT_ROOT]..."
       ㆍ상품그룹 - 신상품 : $grp="new"; 베스트 : $grp="best"; 인기 : $grp="popular"; 추천 : $grp="recom"; 세일 : $grp="sale";
