페이지접근권한

홈 > 환경설정 > 페이지접근권한

페이지접근권한 생성코드

<?
$level = 3;
$msg = "접근권한이 없습니다.";
$backurl = "/";
include "$_SERVER[DOCUMENT_ROOT]/admin/module/levelcheck.php";
?>
생성코드복사

체크사항 (사이트의 특정페이지를 회원 권한별로 접근할 수 있게하는 기능을 제공합니다.)

  • - 레벨별 접근권한 기능외에 로그인 체크기능으로도 이용할 수 있습니다.
  • - $level : 접근권한 레벨을 지정합니다. ( 회원관리>회원등급>등급레벨 값을 입력합니다. ) 3을 입력한경우 3과 같거나 작은경우 페이지 접근이 가능합니다.
  • - $msg : 권한이 없을경우 경고 메세지를 설정합니다. 입력하지 않는경우 기본은 "권한이 없습니다." 입니다.
  • - $backurl : 권한이 없어 경고창이 뜬경우 이동할 페이지를 입력합니다. 기본은history.back(); 입니다.
